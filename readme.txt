Advertising management of the VK AdNetwork.

== Description ==

If you have a site on the WordPress engine and you want to place ads on it and earn money, then with this plugin you will be very easy to do it.

## Use of a 3rd Party or external service

- link to the service: [ads.vk.com/partner](https://ads.vk.com/partner);
- link to the servicesâ€™ a terms of use and/or privacy policies: [terms-for-residents](https://help.mail.ru/legal/terms/adsvk/partners-doc-terms-for-residents);
- The plugin uses the VK Advertising Network API from the domain [ads.vk.com](https://ads.vk.com/), receiving ads from a domain [ad.mail.ru](https://ad.mail.ru/).

### Core Features Overview

* Auto-inject ads via placements
* Use shortcodes to place ads manually in post content
* Show ads on the sidebar and in widgets
* Disable all ads on specific pages
* See you graphs with the number of impressions and earnings of each ad block, you can customize and manage the blocks, timing when they work

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'vk adnetwork'
3. Click 'Install Now'
4. Activate VK Adnetwork on the Plugin dashboard
5. Go to the settings page
6. Insert your `client_id` and `client_secret`

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `vk-adnetwork.zip` from your computer
4. Click 'Install Now'
5. Activate VK Adnetwork in the Plugin dashboard
6. Go to the settings page
7. Insert your `client_id` and `client_secret`

= Using FTP =

1. Download `vk-adnetwork.zip`
2. Extract the `vk-adnetwork` directory to your computer
3. Upload the `vk-adnetwork` directory to the `/wp-content/plugins/` directory
4. Activate VK Adnetwork in the Plugin dashboard
5. Go to the settings page
6. Insert your `client_id` and `client_secret`
